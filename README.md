## 介绍
这是我在学习Vue过程中做的一个UI框架，希望对你有用

## 开始使用

1. 安装框架
   ```
   npm i --save nick-vue-wheels
   ```
2. 引入框架
    ```javascript
    import { Button } from 'nick-vue-wheels'
    import 'nick-vue-wheels/dist/index.css'
    
    export default {
      name: 'app',
      components: {
      'my-button': Button
      }
    }
    ```
